## Install

```bash
$ npm install --save-dev stylelint-config-fresh
```

## Usage

In `.stylelintrc`:

```json
{
  "extends": "stylelint-config-fresh"
}
```

In `package.json`:

```json
{
  "stylelint": {
    "extends": "stylelint-config-fresh"
  }
}
```
